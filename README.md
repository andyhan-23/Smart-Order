# Table Order - 모바일 주문을 통한 테이블 오더 시스템

## 프로젝트 소개
식당에서 고객이 본인의 스마트폰으로 직접 메뉴를 주문하는 테이블 오더 시스템을 만들었습니다.
<br />
이 시스템을 사용하면 업주는 태블릿 설치형 테이블 오덜에 비해 연간 450만 원 이상의 렌탈비를 점갈할 수 있으며, 기기 관리를 매일 해야 하는 번거로움에서도 해방될 수 있습니다.
